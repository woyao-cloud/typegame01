作为产品经理打算使用claude code 生成一个儿童打字游戏网页typescript + html，请生成满足以下要求的CLAUDE.md 及相关的产品文档，以便需求分析员根据你的输出进行需求分析工作
1.这个游戏可以配置练习的字库，字库可以保存到本地暂时不用联网；
2.默认字库是所有键盘上的阿拉伯字母默认练习小写字母，如果是其他字库大小写则根据导入文章实际大小写；
3.默认阿拉伯字母字库，通过控制字母在键盘上的相邻性大小表示难易等级，比如练习s 难度等级低时同屏幕只出现a,w,d,x，随着难度增加则可出现其他位置的字母
4.每局游戏可以配置单词主题，比如小动物主题（青蛙,兔子，猫猫，小狗等）；鲜花主题（向日葵，郁金香，太阳花），其他主题可以参考深圳市小学英语一年级和小学英语二年级教材里出现的名词，难度等级低时只要求敲击一个字母匹配，随着难度的增加要求敲击两个以上字母匹配，直到完全匹配
5.每局打字时间可以配置，比如每局2分钟、5分钟、10分钟，打字速度及正确率按局计算
6 打字速度可以配置，比如初学者每20秒打一个字母
7.通过使用生动的卡通形象比如青蛙增加游戏的趣味性，图片要求大一点，可以参考https://zh.freepik.com/premium%E4%BA%BA%E5%B7%A5%E6%99%BA%E8%83%BD%E5%9C%96%E5%83%8F/close-up-bunch-stuffed-animals-with-different-designs_367491385.htm#fromView=keyword&page=1&position=20&uuid=25cce553-573e-424e-916f-6b62bc469198&query=%E5%8D%A1%E9%80%9A%E5%8A%A8%E7%89%A9%E5%BD%A2%E8%B1%A1
8.尽量提高该游戏的趣味性
