## 1. 产品概述

### 1.1 产品定位
一款面向儿童的英语打字练习游戏，通过趣味性的卡通形象和游戏化设计，帮助儿童在玩耍中熟悉键盘布局、练习打字技能。

### 1.2 核心价值
- **趣味性**: 卡通形象、动画效果、主题场景让打字练习不再枯燥
- **渐进式学习**: 从单字母到完整单词，从相邻键到全键盘
- **可配置性**: 家长/老师可根据孩子水平自定义练习内容
- **离线使用**: 无需联网，保护儿童视力健康

## 2. 功能需求

### 2.1 字库配置系统 (FR-001)

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 - 核心功能 |
| 用户故事 | 作为家长，我希望可以自定义练习的字库，以便针对孩子的薄弱环节进行练习 |

**功能描述**:
- 支持创建、编辑、删除多个字库
- 字库数据保存到本地 (localStorage/IndexedDB)
- 支持字库导入/导出 (JSON 格式)
- 默认字库：26 个小写英文字母

**业务规则**:
- FR-001-R1: 默认字库为 26 个小写字母 (a-z)
- FR-001-R2: 自定义字库可包含任意字符，大小写保留
- FR-001-R3: 字库名称不能为空，最大长度 50 字符
- FR-001-R4: 单个字库最多包含 500 个字符

**验收标准**:
- [ ] 可创建新字库并保存
- [ ] 可编辑已有字库内容
- [ ] 可删除字库 (需确认)
- [ ] 可导出字库为 JSON 文件
- [ ] 可导入 JSON 字库文件

### 2.2 难度系统 (FR-002)

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 - 核心功能 |
| 用户故事 | 作为家长，我希望可以设置游戏难度，以便匹配孩子当前的打字水平 |

**功能描述**:
- 难度等级：⭐简单 / ⭐⭐中等 / ⭐⭐⭐困难
- 难度基于键盘相邻性算法
- 动态调整屏幕出现的字母范围

**难度规则**:

| 难度 | 字母范围 | 说明 |
|-----|---------|------|
| ⭐简单 | 目标字母的相邻键 (上下左右) | 如目标为's'，只出现 a,w,d,x |
| ⭐⭐中等 | 目标字母的同手区键位 | 如目标为's'，可出现左手区域字母 |
| ⭐⭐⭐困难 | 全键盘随机 | 26 个字母随机出现 |

**键盘相邻性定义**:
```
QWERTY 键盘布局:
Row1: q w e r t y u i o p
Row2:  a s d f g h j k l
Row3:   z x c v b n m

相邻键示例 (以 s 为中心):
- 上：w, e
- 下：z, x
- 左：a
- 右：d
```

**验收标准**:
- [ ] 简单难度下只显示相邻字母
- [ ] 中等难度扩大字母范围
- [ ] 困难难度显示全部字母
- [ ] 难度切换即时生效

### 2.5 打字速度配置 (FR-005)

| 属性 | 描述 |
|-----|------|
| 优先级 | P1 - 重要功能 |
| 用户故事 | 作为初学者，我希望字母下落速度慢一些，以便有足够时间反应 |

**验收标准**:
- [ ] 可选择不同速度等级

### 2.6 游戏核心玩法 (FR-006)

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 - 核心功能 |
| 用户故事 | 作为孩子，我希望通过敲击键盘来匹配屏幕上的字母，以便完成游戏 |

**功能描述**:
- 字母/单词从屏幕上方下落
- 玩家敲击对应键盘按键
- 正确输入：字母变色/消失，得分 +1
- 错误输入：震动反馈，不计分
- 超时未输入：判定为 missed

**计分规则**:
- 正确输入：+10 分
- 连续正确：连击 bonus (+5 分/连击)
- 错误输入：0 分
- 超时未输入：-5 分

**验收标准**:
- [ ] 键盘输入正确识别
- [ ] 大小写敏感 (根据字库配置)
- [ ] 实时显示得分
- [ ] 连击系统正常工作

### 2.7 结果统计 (FR-007)

| 属性 | 描述 |
|-----|------|
| 优先级 | P1 - 重要功能 |
| 用户故事 | 作为家长，我希望看到孩子的打字统计，以便了解进步情况 |

**历史记录**:

**验收标准**:
- [ ] 游戏结束显示统计面板
- [ ] WPM 计算正确
- [ ] 正确率计算正确
- [ ] 历史记录可查询

| 属性 | 描述 |

## 3. 非功能需求

### 3.1 性能要求
- 首屏加载时间 < 2 秒
- 键盘响应延迟 < 50ms
- 动画帧率 ≥ 60fps

### 3.2 兼容性要求
- 支持 Chrome 90+
- 支持 Firefox 88+
- 支持 Safari 14+
- 支持 Edge 90+

### 3.3 可用性要求
- 无需联网即可使用
- 数据本地持久化
- 支持 PWA 离线使用

### 3.4 安全要求
- 无用户账号系统
- 无网络传输
- 无敏感数据存储

## 4. 数据字典

### 4.1 字库数据结构
```typescript
interface WordLibrary {
  id: string;
  name: string;
  characters: string[];  // 字符列表
  isDefault: boolean;
  createdAt: number;
  updatedAt: number;
}
```

### 4.2 游戏配置结构
```typescript
interface GameConfig {
  libraryId: string;      // 字库 ID
  themeId: string;        // 主题 ID
  difficulty: 1 | 2 | 3;  // 难度等级
  duration: number;       // 时长 (秒)
  speed: 1 | 2 | 3 | 4;   // 速度等级
  minMatchLength: number; // 最小匹配长度
}
```

### 4.3 游戏记录结构
```typescript
interface GameRecord {
  id: string;
  config: GameConfig;
  score: number;
  wpm: number;
  accuracy: number;
  maxCombo: number;
  correctCount: number;
  missCount: number;
  timestamp: number;
}
```
