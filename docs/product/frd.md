## 1. 文档概述

### 1.1 产品定位
一款面向儿童的英语打字练习游戏，通过轨道前进式的游戏设计和趣味性的卡通形象，帮助儿童在玩耍中熟悉键盘布局、练习打字技能。

### 1.2 核心价值
- **轨道前进式玩法** - 字母沿云朵轨道移动，增加游戏紧张感
- **渐进式难度** - 从单字母到完整单词，从相邻键到全键盘
- **双模式练习** - 字母练习 / 单词练习 可选
- **中文界面** - 降低儿童和家长的认知负担

### 1.3 功能优先级汇总

| 优先级 | 功能模块 | 数量 |

| P0 | MVP 核心功能 | 11 项 |
| P1 | 扩展功能 | 3 项 |

### FRD-003：难度系统

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 |
| 功能类型 | 核心玩法 |
| 用户故事 | 作为用户，我希望设置游戏难度，以便匹配我的打字水平 |

**功能描述**：
- 三档难度：⭐简单 / ⭐⭐中等 / ⭐⭐⭐困难
- 难度影响出现的字母范围
- 难度影响需要匹配的字母数量

**难度规则表**：

| 难度 | 字母范围 | 匹配长度 | 适用人群 |
|-----|---------|---------|---------|
| ⭐简单 | 目标字母的相邻键 | 首字母 | 零基础 |
| ⭐⭐中等 | 同手区字母 | 前 2-3 字母 | 有基础 |
| ⭐⭐⭐困难 | 全键盘 26 字母 | 完整单词 | 熟练者 |

**键盘相邻性定义**（以 QWERTY 键盘为准）：

```
相邻键示例（以 S 为中心）:
- 上排：W, E
- 下排：Z, X
- 左侧：A
- 右侧：D
- 相邻键集合：[A, W, D, X, Z]

同手区示例（以 S 为中心，左手区）:
- 左手字母：Q, W, E, R, T, A, S, D, F, G, Z, X, C, V, B
```

**功能规则**：
- FRD-003-R1: 简单难度下，只出现目标字母的相邻键
- FRD-003-R2: 中等难度下，出现同手区字母
- FRD-003-R3: 困难难度下，26 个字母随机出现
- FRD-003-R4: 难度选择即时生效，无需重启游戏

**验收标准**：
- [ ] 简单难度只显示相邻字母（如目标 S，只出现 A,W,D,X,Z）
- [ ] 中等难度显示同手区字母
- [ ] 困难难度显示全部字母
- [ ] 难度切换后，新字母立即按新规则生成

### FRD-005：匹配逻辑

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 |
| 功能类型 | 核心玩法 |
| 用户故事 | 作为用户，我希望通过键盘输入匹配字母，以便完成游戏 |

**功能描述**：
- 根据难度等级，需要匹配的字母数量不同
- 实时显示已输入字符的匹配状态
- 正确输入后字母消失并计分

**匹配规则表**：

| 难度 | 目标示例 | 需要输入 | 匹配成功条件 |
|-----|---------|---------|-------------|
| ⭐简单 | **F**rog | F | 输入首字母 |
| ⭐⭐中等 | **FR**og | FR | 输入前 2 字母 |
| ⭐⭐⭐困难 | **FROG** | FROG | 输入完整单词 |

**匹配反馈**：
```
目标：FROG
输入 F → 显示 F**ROG** (已输入高亮)
输入 FR → 显示 FR**OG**
输入 FRO → 显示 FRO**G**
输入 FROG → 完整高亮，计分，消失
```

**功能规则**：
- FRD-005-R1: 字母练习模式下，匹配单个字母
- FRD-005-R2: 单词练习模式下，根据难度匹配相应长度
- FRD-005-R3: 输入错误时，字母震动并变红
- FRD-005-R4: 匹配成功后，目标消失并计分

**验收标准**：
- [ ] 字母模式下单次按键即可匹配
- [ ] 单词模式下按难度匹配相应长度
- [ ] 已输入字符正确高亮显示
- [ ] 输入错误时字母震动变红
- [ ] 匹配成功后目标消失

### FRD-006：计分系统

| 属性 | 描述 |
|-----|------|
| 优先级 | P0 |
| 功能类型 | 核心玩法 |
| 用户故事 | 作为用户，我希望看到我的得分，以便了解我的表现 |

**功能描述**：
- 正确输入得分
- 连续正确获得连击奖励
- 超时未输入扣分

**计分规则**：
| 事件 | 分数变化 |
|-----|---------|
| 正确输入 | +10 分 |
| 连击奖励（5 连击以上） | +5 分/连击 |
| 错误输入 | 0 分 |
| 超时未输入 (Miss) | -5 分 |

**连击规则**：
| 连击数 | 效果 |
|-------|------|
| 5 连击 | "Good!" 提示 |
| 10 连击 | "Great!" 提示 |
| 20 连击 | "Amazing!" 提示 |
| 50 连击 | "Perfect!" 提示 + 特殊效果 |

**功能规则**：
- FRD-006-R1: 基础得分为每次正确输入 +10 分
- FRD-006-R2: 连击从 5 开始触发奖励，每连击额外 +5 分
- FRD-006-R3: 错误输入或 Miss 会重置连击计数
- FRD-006-R4: Miss 扣 5 分，最低为 0 分（不出现负分）

**验收标准**：
- [ ] 正确输入时分数正确增加
- [ ] 连击数正确显示和计算
- [ ] 连击奖励分数正确发放
- [ ] Miss 时分数正确扣除
- [ ] 分数不会出现负值

### FRD-010：游戏统计

**统计指标**：
| 指标 | 计算方式 |

```json
{
  "id": "record_001",
  "timestamp": 1711785600000,
  "mode": "word",
  "difficulty": 2,
  "score": 350,
  "wpm": 25,
  "accuracy": 0.92,
  "maxCombo": 15,
  "correctCount": 46,
  "errorCount": 4,
  "missCount": 2
}
```

**功能规则**：
- FRD-010-R1: 游戏结束显示统计面板
- FRD-010-R2: WPM 计算公式为 (正确字符数/5) / 分钟数
- FRD-010-R3: 正确率保留两位小数（百分比格式）
- FRD-010-R4: 保存最近 100 条游戏记录

**验收标准**：
- [ ] 游戏结束显示完整统计面板
- [ ] WPM 计算正确
