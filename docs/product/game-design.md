# 游戏设计文档

## 儿童打字游戏

---

## 1. 游戏概述

### 1.1 游戏类型
教育类打字练习游戏 (Educational Typing Game)

### 1.2 目标受众
- **主要用户**: 6-10 岁儿童 (小学一、二年级)
- **次要用户**: 家长、英语老师

### 1.3 游戏目标
通过趣味性的游戏方式，帮助儿童：
- 熟悉 QWERTY 键盘布局
- 提高打字速度和准确性
- 学习英语单词拼写

---

## 2. 核心游戏机制

### 2.1 字母下落系统

```
┌─────────────────────────────────┐
│     🐸                          │
│         [S]                     │  ← 下落的字母
│                                 │
│    ┌─────┬─────┬─────┐          │
│    │  A  │  S  │  D  │          │  ← 键盘提示区
│    └─────┴─────┴─────┘          │
│                                 │
│  得分：150   连击：5   ⏱️ 1:30   │
└─────────────────────────────────┘
```

**下落规则**:
- 字母从屏幕顶部随机位置生成
- 匀速向下移动
- 到达底部前未输入则判定为"miss"
- 输入正确后字母消失并计分

### 2.2 键盘相邻性算法

**相邻键定义**:
```javascript
// 键盘布局映射
const keyboardLayout = [
  ['q', 'w', 'e', 'r', 't', 'y', 'u', 'i', 'o', 'p'],
  ['a', 's', 'd', 'f', 'g', 'h', 'j', 'k', 'l'],
  ['z', 'x', 'c', 'v', 'b', 'n', 'm']
];

// 获取相邻键
function getAdjacentKeys(key, difficulty) {
  const position = findKeyPosition(key);

  if (difficulty === 1) { // 简单：只包含直接相邻
    return getDirectNeighbors(position);
  } else if (difficulty === 2) { // 中等：同手区
    return getSameHandZone(position);
  } else { // 困难：全键盘
    return getAllKeys();
  }
}
```

**难度映射表**:

| 目标键 | 简单难度 (相邻) | 中等难度 (同手区) | 困难难度 |
|-------|---------------|----------------|---------|
| S | a, w, d, x, z | q,w,e,a,s,d,f,z,x,c | 全键盘 |
| J | h, u, i, k, m | y,u,i,o,p,h,j,k,l,n,m | 全键盘 |
| A | q, s, z | q,w,e,a,s,d,f,z,x,c | 全键盘 |

### 2.3 匹配系统

**匹配长度规则**:

| 难度 | 单词示例 | 需要输入的字母 |
|-----|---------|--------------|
| 简单 | **f**rog | f (首字母) |
| 中等 | **fr**og | fr (前 2 字母) |
| 困难 | **frog** | frog (完整单词) |

**匹配反馈**:
```
输入 "f" → 显示 "f**rog" (已输入高亮)
输入 "fr" → 显示 "fr**og"
输入 "frog" → 完整高亮，计分，消失
```

## 3. 主题系统设计

### 3.1 主题列表

| ID | 主题名称 | 主色调 | 卡通形象 | 词汇示例 |
|----|---------|-------|---------|---------|
| animals | 🐸 小动物 | 绿色 | 青蛙、兔子、猫猫、小狗 | frog, cat, dog |
| flowers | 🌻 鲜花 | 粉色 | 向日葵、郁金香 | flower, rose |
| fruits | 🍎 水果 | 橙色 | 苹果、香蕉 | apple, banana |
| vehicles | 🚗 交通工具 | 蓝色 | 小汽车、公交车 | car, bus |
| school | 🏫 学校用品 | 紫色 | 铅笔、书本 | pen, book |
| family | 👨‍👩‍👧 家庭成员 | 暖黄 | 家庭卡通形象 | mom, dad |
| colors | 🌈 颜色 | 彩虹 | 彩虹精灵 | red, blue |
| numbers | 🔢 数字 | 多彩 | 数字小人 | one, two |

### 3.2 主题视觉规范

**背景设计**:
- 每个主题有独特的背景图案
- 使用柔和、不刺眼的颜色
- 背景透明度 80%，确保文字清晰

**卡通形象规格**:
- 主形象尺寸：256x256px
- 动作状态：idle, jump, celebrate, encourage
- 格式：PNG (透明背景) 或 SVG

## 4. 趣味性设计

### 4.1 即时反馈系统

**正确输入**:
- ✅ 绿色闪光效果
- 🔊 清脆的"叮"声
- 🎯 卡通形象跳跃动画
- 💫 粒子效果 (可选)

**错误输入**:
- ❌ 红色震动效果
- 🔊 低沉的"嘟"声
- 💭 鼓励提示："Try again!"

**连击奖励**:
| 连击数 | 效果 |
|-------|------|
| 5 连击 | "Good!" 提示 |
| 10 连击 | "Great!" + 闪光 |
| 20 连击 | "Amazing!" + 烟花 |
| 50 连击 | "Perfect!" + 特殊动画 |

### 4.2 成就系统

**成就列表**:

| 成就名称 | 达成条件 | 奖励 |
|---------|---------|------|
| 第一次尝试 | 完成第一局游戏 | 🏅 新手徽章 |
| 神枪手 | 单局正确率 100% | 🎯 精准徽章 |
| 快手王 | WPM > 30 | ⚡ 速度徽章 |
| 坚持就是胜利 | 连续 7 天登录 | 📅 坚持徽章 |
| 单词大师 | 完成所有主题 | 🎓 大师徽章 |

### 4.3 动画效果清单

| 场景 | 动画类型 | 持续时间 |
|-----|---------|---------|
| 字母生成 | fade in + scale | 0.3s |
| 字母下落 | translate Y | 持续 |
| 正确输入 | flash green + pop | 0.5s |
| 错误输入 | shake red | 0.3s |
| 连击 | combo counter scale | 0.2s |
| 游戏结束 | slide up panel | 0.5s |
| 升级 | confetti burst | 2s |

## 5. 难度曲线设计

### 5.1 难度 progression

```
新手阶段 (Level 1-3)
├── 简单难度
├── 单字母匹配
├── 超慢速度
└── 小动物主题 (熟悉词汇)

进阶阶段 (Level 4-6)
├── 中等难度
├── 双字母匹配
├── 慢速
└── 更多主题解锁

挑战阶段 (Level 7-9)
├── 困难难度
├── 完整单词匹配
├── 中速
└── 所有主题

大师阶段 (Level 10+)
├── 困难难度
├── 完整单词匹配
├── 快速
└── 自定义字库
```

### 5.2 自适应难度建议

```javascript
// 根据玩家表现动态调整
function adjustDifficulty(stats) {
  if (stats.accuracy > 0.9 && stats.wpm > 20) {
    return increaseDifficulty();
  } else if (stats.accuracy < 0.5 || stats.wpm < 10) {
    return decreaseDifficulty();
  }
  return currentDifficulty;
}
```
