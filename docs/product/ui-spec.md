## 4. 游戏界面 (GameScreen)

### 4.1 布局设计

```
┌─────────────────────────────────────────────────────────┐
│  ⏱️ 1:45    🎯 250    🔥 连击 x7    ❤️ 正确率 92%      │ ← 顶部状态栏
│  [⏸️ 暂停] [🏠 退出]                                    │ ← 右上角控制按钮
├─────────────────────────────────────────────────────────┤
│                                                         │
│     ☁️        🎈      ☁️        🎈      ☁️            │ ← 背景装饰
│                                                         │
│    ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐  ┌─────┐       │
│    │ 🎈  │  │ 🎈  │  │ 🎈  │  │ 🎈  │  │ 🎈  │       │ ← 5 条轨道
│    │ FR  │  │ CA  │  │ D   │  │ TR  │  │ BI  │       │   每轨道 1 个单词
│    └─────┘  └─────┘  └─────┘  └─────┘  └─────┘       │
│                                                         │
│  ┌─────────────────────────────────────────────────┐   │
│  │  当前单词提示 (底部中央)                         │   │
│  │     F**ROG**    **CA**T    **D**OG              │   │
│  │     (已输入绿色，待输入黄色高亮)                 │   │
│  └─────────────────────────────────────────────────┘   │
│                                                         │
│  云朵背景 + 星星装饰 + 主题吉祥物漂浮                   │
├─────────────────────────────────────────────────────────┤
│  天空主题，渐变背景                                      │
└─────────────────────────────────────────────────────────┘
```

**设计说明**：
- 字母/单词显示在卡通形象（气球🎈）上方，形成"卡通形象携带单词"的视觉效果
- 5 条轨道垂直排列，每条轨道最多显示 1 个活跃单词
- 单词从轨道顶部（🏁旗帜下方）开始下落，进度从 0% 到 100%
- 当下落进度达到 100% 时，字母消失并判定为 Miss
- 底部中央显示所有活跃单词的匹配进度
- **移除键盘提示区**（简化界面，专注于单词匹配）

### 4.1.1 下落动画规范

| 属性 | 值 | 说明 |
|-----|-----|------|
| 起始位置 | progress=0% | 轨道顶部（🏁旗帜下方） |
| 结束位置 | progress=100% | 轨道底部 |
| 下落速度 | 5 档可调 | 超慢 (30s) / 慢 (20s) / 正常 (10s) / 快速 (5s) / 极速 (3s) |
| 动画类型 | 匀速下落 | linear 缓动 |
| 渲染方式 | requestAnimationFrame | 60fps 游戏循环驱动 |

### 4.2 组件清单

| 组件名 | 类型 | 描述 |
|-------|------|------|
| StatusBar | Component | 顶部状态栏 |
| Timer | Display | 倒计时显示 |
| Score | Display | 分数显示 |
| Combo | Display | 连击显示 |
| Accuracy | Display | 正确率显示 |
| ControlButtons | Component | 暂停/退出按钮 (右上角) |
| TrackContainer | Component | 轨道容器 (5 条垂直轨道) |
| WordBalloon | Component | 卡通气球 + 单词组合 |
| ActiveWordHint | Component | 底部当前单词提示 |
| ComboOverlay | Component | 连击覆盖层 (中央) |

### 4.3 状态显示

| 状态项 | 格式 | 更新频率 |
|-------|------|---------|
| 时间 | MM:SS | 每秒 |
| 分数 | 数字 | 实时 |
| 连击 | xN | 实时 |
| WPM | 数字 | 每次输入后 |
| 正确率 | N% | 每次输入后 |

### 4.4 交互行为

| 元素 | 事件 | 行为 |
|-----|------|------|
| 键盘输入 | KeyDown | 处理字母匹配 |
| 字母匹配成功 | - | 已输入部分变绿色，播放音效 |
| 字母匹配失败 | - | 单词震动变红 |
| 时间归零 | - | 自动结束游戏 |
| ESC 键 | KeyDown | 暂停游戏 |
| 暂停按钮 | Click | 暂停游戏，显示暂停菜单 |
| 退出按钮 | Click | 弹出确认对话框 |

### 4.5 视觉反馈

| 事件 | 效果 | 持续时间 |
|-----|------|---------|
| 正确输入 | 已输入字母变浅灰色 + 半透明 (#9ca3af, opacity-40) | 持续显示 |
| 错误输入 | 单词震动 + 红色边框 | 0.3s |
| 连击达成 (≥5) | 中央火焰图标 + 数字脉动 | 2s |
| 连击 (≥10) | 火焰动画增强 | 持续显示 |
| 单词匹配成功 | 渐隐 + 星星粒子 | 0.3s |
| Miss | 灰色下沉 | 0.5s |
| 时间警告 (<30s) | 倒计时变红色 + 闪烁 | 持续显示 |

### 4.6 单词匹配进度显示

| 部分 | 颜色 | 透明度 | 说明 |
|-----|------|--------|------|
| 已输入 | #9ca3af (浅灰色) | opacity-40 | 已正确匹配的字母，渐隐效果 |
| 待输入 | #fbbf24 (黄色高亮) | 等待输入的字母 |
| 底部提示 | #0ea5e9 (蓝色边框) | 当前活跃单词的匹配进度 |

### 4.7 暂停/退出控制

| 按钮 | 位置 | 行为 |
|-----|------|------|
| 暂停按钮 (⏸️/▶️) | 右上角 | 暂停/恢复游戏 |
| 退出按钮 (🏠) | 右上角 (暂停右侧) | 弹出确认对话框 |

### 4.8 连击覆盖层

当连击 ≥ 5 时显示在屏幕中央：

```
        🔥
       x15
      连击!
    ⭐ ✨ ⭐
```

| 元素 | 动画 | 说明 |
|-----|------|------|
| 火焰 emoji | 弹跳动画 | 连击视觉符号 |
| 连击数字 | 脉动 + 火焰色 | 当前连击数 |
| 星星装饰 | 闪烁动画 | 增强视觉效果 |
